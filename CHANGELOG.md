# 変更履歴

## Ver.3.1.0 (2026-07-18)

- 店舗比較画面の初期表示を「入力・編集／日別」に変更
  - 上部の「比較」ボタンから入ったときのみ固定。比較画面内でのタブ切り替えではリセットされません
- 売上管理と店舗比較のバックアップCSVを統合
  - 形式: `データ種別,キー,店舗ID,値1〜値10`(UTF-8 / BOM付き)
  - データ種別: META(バージョン・出力日時) / SETTING(店舗名) / SALES(売上) / HIKAKU_MONTH(比較・月次) / HIKAKU_DAY(比較・日別)
- 両画面上部の共通ヘッダーにバックアップボタン(💾)を追加
  - どちらの画面からでも同じバックアップメニューが開きます
  - ファイル名例: sales-management-backup_2026-07-18_1300.csv
- CSVインポート時の重複登録防止と安全な復元処理を追加
  - 統合CSVは「置き換え方式」: 復元前に確認ダイアログを表示し、検証に成功した場合のみ既存データを置き換え
  - 検証失敗・キャンセル時は既存データを一切変更しません
  - 旧形式CSV(売上管理CSV / 店舗比較CSV)もバックアップメニューから読み込み可能(従来どおり日付単位の上書きマージ)

### 既知の制限
- PINコードはセキュリティ上の理由で統合CSVに含まれません(復元後も現在のPINが維持されます)
- 統合CSVのインポートは置き換え方式のため、復元後にバックアップ以降の入力分は消えます。復元前に最新のバックアップを取ることを推奨します
